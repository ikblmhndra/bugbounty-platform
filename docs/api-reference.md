# API Reference

Base URL (default): `http://localhost:8000/api/v1`

Interactive docs:

- Swagger UI: `http://localhost:8000/docs`
- ReDoc: `http://localhost:8000/redoc`

## Health

```bash
curl http://localhost:8000/health
```

## Scans

### Create scan

```bash
curl -X POST http://localhost:8000/api/v1/scans \
  -H "Content-Type: application/json" \
  -d '{
    "domain": "example.com",
    "options": {
      "run_ffuf": false,
      "run_gowitness": true,
      "nuclei_severity": "medium,high,critical"
    }
  }'
```

### List scans

```bash
curl "http://localhost:8000/api/v1/scans?limit=50&offset=0"
```

### Get scan by id

```bash
curl http://localhost:8000/api/v1/scans/<scan_id>
```

### Get scan logs

```bash
curl "http://localhost:8000/api/v1/scans/<scan_id>/logs?limit=100"
```

### Cancel scan

```bash
curl -X DELETE http://localhost:8000/api/v1/scans/<scan_id>
```

## Findings

### List findings

```bash
curl "http://localhost:8000/api/v1/findings?scan_id=<scan_id>"
```

### Get validation suggestions

```bash
curl http://localhost:8000/api/v1/findings/<finding_id>/validate
```

## Reports

### Generate report

```bash
curl "http://localhost:8000/api/v1/reports/<scan_id>?fmt=json"
curl "http://localhost:8000/api/v1/reports/<scan_id>?fmt=markdown"
curl "http://localhost:8000/api/v1/reports/<scan_id>?fmt=html" > report.html
```

## Dashboard and Misc

```bash
curl http://localhost:8000/api/v1/dashboard
curl http://localhost:8000/api/v1/assets
curl http://localhost:8000/api/v1/paths
```

## Notes

- Request and response schemas are defined in `app/schemas/schemas.py`.
- Endpoint implementations live under `app/api/`.
- For exact field contracts, rely on OpenAPI docs generated by FastAPI.
